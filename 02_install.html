Instalação

Utilize o composer para começar seu projecto.

Requisitos do servidor

PHP >= 5.6.4

PDO PHP Extension

Instalação via Composer

Para instalar uma nova aplicação Syph, precisaremos do composer, caso você, não o tenha instalado, clique no link para saber mais, porém recomendamos a instalação do composer de forma global

Download Composer

Com o composer instalado, execute o comando:


composer create project syph-fw/syph-app project

Quando o composer terminar a instalação, basta acessar o seus endereço localhost. localhost/project/public/índex.php e a primeira tela será retornada.
