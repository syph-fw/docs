Minha primeira View

Na página "Meu Primeiro Controller" renderizamos uma View simples para mostrar a integração da camada de controle com a visão, agora vamos melhorar a View anterior, adicionando parâmetros passados pelo Controller para exemplificar situações comuns na criação de aplicações WEB.


//TestController.php
public function test(){
$title = "Teste de View";
return View::render($this->createView('DemoApp:test.php'),array("title" => $title));
}

Como citado na pagina "Meu Primeiro Controller" as chamadas à uma View podem receber um array associativo com as variáveis que se desejam ser acessadas no arquivo de visão, como no exemplo abaixo, a variável $title sendo acessada nos arquivos de visão.


//test.php
<html>
<head>
<title><?php echo $title; ?></title>
</head>
<body>
<h1><?php echo $title; ?></h1>
</body>
</html>


//test.html.twig
<html>
<head>
<title>{{ "{{ title" }} }}</title>
</head>
<body>
<h1>{{ "{{ title" }} }}</h1>
</body>
</html>

O syph-framework aceita a forma padrão de view em arquivos .php e em arquivos da template engine Twig, .twig.
